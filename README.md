# AI Feedback Analyzer

## Problem

Customer feedback often gets collected but not properly analyzed, causing valuable product insights to be missed.

## Solution

Built an AI-powered automation workflow that collects customer feedback, summarizes it using Gemini AI, and stores structured insights in Notion automatically.

## Workflow

Google Forms → Make.com → Gemini AI → Notion

## What it does

* Captures customer reviews automatically
* Uses AI to summarize feedback
* Extracts key insights
* Stores structured summaries in Notion
* Reduces manual review effort

## Tech Stack

* Make.com
* Google Forms
* Gemini AI
* Notion

## Features

* Real-time feedback collection
* AI summarization
* Centralized knowledge base
* No-code automation
